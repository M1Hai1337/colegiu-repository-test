Essential Git Commands

Clone a repository

git clone <url>

Download a remote repository to your computer so you can work on it locally.

Check status

git status

See which files have been changed, added, or are ready to be committed.

Stage changes

git add .

Add all modified files to the staging area, preparing them for a commit.

Commit changes

git commit -m "message"

Save a snapshot of your staged changes with a descriptive message.

Push to remote

git push

Upload your local commits to the remote repository on GitHub.

Pull updates

git pull

Download and merge the latest changes from the remote repository.

Create a branch

git checkout -b <name>

Create a new branch and switch to it. Useful for working on features separately.

Switch branch

git checkout <name>

Switch to an existing branch to continue working on it.

View history

git log --oneline

Show a compact list of all previous commits in the current branch.

Typical Workflow

How to save and upload your changes:

git add .
→
git commit -m "msg"
→
git push
Tip: Always run git pull before you start working to make sure you have the latest version of the project!

Student Tasks

  1. Task 1 — Clone the repository Open your terminal and run git clone <repo-url> to download this project to your computer.
  2. Task 2 — Create your own branch Create a new branch with your name: git checkout -b feature/YourName
  3. Task 3 — Add yourself to the students list Open index.html, scroll to the "Students" section, and add a new card with your name inside the student-grid div.
  4. Task 4 — Stage & commit your changes Run git add . then git commit -m "Add YourName" to save your work.
  5. Task 5 — Push your branch to GitHub Upload your branch: git push -u origin feature/YourName
  6. Task 6 — Create a Pull Request Go to the repository on GitHub, click "Compare & pull request", add a description, and submit it.
  7. Task 7 — Review a classmate's Pull Request Open a classmate's PR on GitHub, read their changes, and leave a comment or approve it.
